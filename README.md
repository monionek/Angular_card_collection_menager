Card Collection Manager

A web application for managing and organizing a personal card collection.

Tech Stack

Frontend: Angular 21

Backend: JSON Server (mock REST API)

Getting Started
1. Install dependencies
npm install
2. Start the backend
npm run backend
3. Start the frontend (Angular app)
npm run start

The application should now be available in your browser (default Angular dev server URL).

Features

Add and manage cards in your collection

Browse and organize items

Simple full-stack setup for local development

Development Notes

Make sure the backend is running before starting the frontend to ensure API requests work correctly.
